# Notes for running terraform in VS Code

- cd .
- az login 
- az account set --subscription "[Your sub id]"
- terraform init
- terraform validate
- terraform refresh -var-file="local.tfvars"
- terraform plan -var-file="local.tfvars"
- terraform apply -var-file="local.tfvars"
